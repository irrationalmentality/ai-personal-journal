# AI Personal Journal *(working title)*

A journaling app built on a simple, contrarian idea: **you should be able to run on your own calendar, not the world's.** You declare a personal epoch — "my time starts now" — and your own units of time (your own week, your own year), and an AI works to keep that alternative sense of time alive and meaningful.

This is a portfolio project exploring how to integrate AI both *into the product* and *into the development process* — deliberately not another mainstream docs bot.
